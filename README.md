## Features
- Create account;
- Search content by word;
- List content by genre;
- Access content on YouTube.

## Device Compatibility
- Mobile (max-width:570px);
- iPad (1024x768);
- Laptop (1440x900);
- Desktop (1680x1050);
- Desktop (1920x1080);
- Desktop (min-width:2560px).
